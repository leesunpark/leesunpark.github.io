Paik/Abe Video Synthesizer, Willy-Nilly Version

This project generates NTSC video using ATMEL’s ATmega16 microcontroller.

Development Environment

macOS

Install CrossPack for AVR https://www.obdev.at/products/crosspack/download.html

Windows

Install Win-AVR https://sourceforge.net/projects/winavr or

Install Atmel Studio http://www.atmel.com/tools/atmelstudio.aspx#download or

Install CodeVisionAVR http://hpinfotech.ro/cvavr_download.html

Linux

1. Install Code::Blocks

> sudo apt-get update

> sudo apt-get install codeblocks

2. Install AVRDude

> sudo add-apt-repository ppa:pmjdebruijn/avrdude-release

> sudo apt-get update

> sudo apt-get install binutils-avr gcc-avr avr-libc avrdude libc6-dev-i386

How-to

1. Simply purchase this product(then go to 3) or build the circuit from scratch.

2. Load the binary (*.hex) to the flash memory of the ATmega16 micro controller.

3. Connect the composite video cable to the TV.

4. Connect the audio cable to the audio device.

5. Control video signals using knobs.

6. Circuit-bend or edit this source code if you come up with an idea.

7. Enjoy.
